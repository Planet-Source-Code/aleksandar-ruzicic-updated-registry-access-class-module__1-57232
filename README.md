## \[Updated\]Registry Access Class Module


</div>

### Description

With this class module you can access windows's registry base and edit String, DWORD or Binary data type.

List of functions (in class):

-ReadString

-WriteString

-ReadDWORD

-WriteDWORD

-ReadBinary

-WriteBinary

-KillValue

-ValueExists

-EnumValues

-KeyExists

-CreateKey

-KillKey

-EnumKeys

-HasSubKeys

-ExportToReg

-ImportFromReg

-isBinValid

Enjoy!
